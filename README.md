AI Resume Analyzer

AI Resume Analyzer is a Streamlit-based web application that analyzes resumes and provides ATS scores, skill gap analysis, role-based recommendations, and AI-powered career suggestions using Google's Gemini API.

Features

- Resume PDF Upload
- Resume Text Extraction
- Skill Detection
- ATS Score Calculation
- Missing Skills Analysis
- Role-Based Skill Gap Analysis
- AI-Powered Suggestions (Gemini)
- Interactive Charts
- Downloadable Resume Report

Tech Stack

- Python
- Streamlit
- PDFPlumber
- Pandas
- Plotly
- Google Gemini API

Installation

git clone <repository-url>
cd AI-Resume-Analyzer
pip install -r requirements.txt
streamlit run app.py

Environment Variables

Create a ".env" file:

GEMINI_API_KEY=your_api_key_here
